#  Live Sales Analytics Dashboard

This is a dedicated, interactive tool designed to give stakeholders quick insights into our sales performance. Built with Streamlit, it effectively transforms raw sales data into immediate, filterable visualizations and key metrics.

The dashboard is structured to answer core business questions by providing:

* **Real-time KPIs:** Instant view of total revenue, orders, and average sale value.
* **Regional Analysis:** Tracking of performance across all key territories.
* **Category Deep Dive:** Clear visibility into which product categories are driving the most revenue.
* **Time Series Trends:** Analysis of monthly sales spikes and dips.

---

##  Built With

* **Streamlit** (Web App Framework)
* **Pandas** (Data Processing)
* **Plotly** (Interactive Charts)

---

## Quick Launch 
To run this dashboard locally, ensure you have the files and run:

1.  **Install Requirements:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Run the App:**
    ```bash
    streamlit run app.py
    ```
